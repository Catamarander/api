Method Summary - TripIt
Object create(String objectType, String divId)
          Creates and returns a new plugin object of the given objectType. Currently the only object type supported is "addToTripIt". The divId argument (optional) corresponds to the id of the div element where the plugin will be rendered, and defaults to 'add-to-tripit-plugin'.
init(Object {api_key, domain})
          Initializes the TripIt SDK object, given the API key and domain registered for the Add-to-TripIt plugin.
Integer initializePlugin(Object plugin, Function callback)
          Initializes the plugin by rendering the IFrame for the plugin. If a callback function is specified, it will be executed when the plugin finishes performing a user-initiated action.
