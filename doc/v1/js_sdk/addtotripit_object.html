Method Summary - AddToTripIt
Object create(String objectType)
          Creates and returns a new trip object of the given objectType. The object types supported are "activity", "address", "lodging", "restaurant", "transport", and "traveler".
Object getPayload()
          Returns the trip object that will be saved in TripIt when the Add-to-TripIt button is clicked.
removePayload()
          Removes any trip object associated with the Add-to-TripIt button.
setPayload(Object tripObject)
          Sets the trip object that will be saved in TripIt when the Add-to-TripIt button is clicked. The tripObject parameter can be of object type Activity, Lodging, Restaurant, or Transport.
